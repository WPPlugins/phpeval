== Description ==

The PHPEval Plugin is a Wordpress plugin which allows users to write php code inside of their pages. Embedded php code will be executed when the post is shown. In doing so, the plugin offers the possibility to utilize dynamic content inside of posts. In contrast to other plugins with the same aim, this on is focus on security.  Here's a features list:

* Easy to setup and configure
* Simple syntax
* Central administration of code
* More coming up soon!

YES, I do consider feature requests, and that is what makes this plugin work. The discussions go here: [Technical Support](http://code.google.com/p/phpeval/ "Technical Support")

== Installation ==

1. Upload archive contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure your settings and check out the Dashboard

== Frequently Asked Questions ==

= Is my php code exposed when I disable the plugin? =
NO your code will remain safe, it´s only the {phpeval=tag} which is show.

= Can I see what pages who uses the phpeval engine? =
Not in this version, but I have it as a request feature, and will be added in next release
